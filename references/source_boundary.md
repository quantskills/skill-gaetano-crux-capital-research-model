# Source Boundary

Allowed sources:

- Public X posts
- Public Substack pages
- Public company filings
- Public earnings transcripts
- Public presentations
- Public technical papers
- User-provided exports or notes

Not allowed unless the user has rights and explicitly provides them:

- Paid Substack articles behind a paywall
- Private Discord, DM, or member-only notes
- Full copied articles from restricted sources
- Broker statements or portfolio screenshots treated as audited proof

Performance claims:

- Treat public return or position claims as unverified unless backed by audited records, broker statements, official competition standings, or inspectable portfolio records.

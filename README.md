# Gaetano Crux Capital Research Model

> Public-material agent skill for photonics, optical networking, Physical AI, and AI infrastructure research.

> 基于公开资料复刻 Gaetano / Crux Capital 的光子、光网络、Physical AI 与 AI 基础设施研究框架。

Language / 语言: [English](#english) | [中文](#中文)

## English

### What This Is

`gaetano-crux-capital-research-model` is a portable agent skill for analyzing public Gaetano / Crux Capital research logic.

It helps an agent turn public X posts, public Substack pages, company coverage notes, filings, earnings transcripts, and technical references into a structured research model.

It does not copy paid content, verify private portfolio returns, or recommend trades.

### Best Use Cases

- Photonics and optical networking thesis decomposition
- AI data-center interconnect and CPO supply-chain mapping
- Physical AI and robotics infrastructure research
- Small-cap AI infrastructure company coverage
- Chokepoint, catalyst, and risk extraction

### Repository Boundary

Commit the skill folder, references, and lightweight Markdown outputs. Do not commit raw exports, paid articles, large CSVs, private notes, or downloaded price histories.

## 中文

### 这是什么

`gaetano-crux-capital-research-model` 是一个通用 Agent 平台 Skill，用来分析 Gaetano / Crux Capital 的公开研究方法。

它适合把公开 X 帖子、公开 Substack 页面、公司覆盖笔记、财报电话会、技术论文和行业资料，整理成结构化研究模型。

它不是跟单工具，不验证私人组合收益，也不复制付费内容。

### 适合做什么

- 拆解 Photonics / 光网络 thesis
- 映射 AI 数据中心互连、CPO、光模块供应链
- 分析 Physical AI、机器人基础设施和传感器链条
- 研究小市值 AI 基础设施公司
- 提取 chokepoint、催化、风险和跟踪指标

### GitHub 边界

建议上传 Skill 文件夹、references 和轻量 Markdown 报告。不要上传原始导出、付费文章、大型 CSV、私人笔记或下载的价格数据。
